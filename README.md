WebGL Text
---------------

An experiment with word art in webgl. Forked from [`webgl-text`](https://github.com/caldera-design/webgl-text).

Installation
============
Use [npm](https://www.npmjs.com/) to install:
```bash
npm install webgl-fancy-text
```
